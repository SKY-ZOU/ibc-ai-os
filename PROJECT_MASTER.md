# IBC AI Trade OS - Project Master

## 1. 项目定义
IBC AI Trade OS 是一个面向全球合作伙伴、以中国为核心交易枢纽的 AI 跨境交易平台。
平台目标是通过 AI 招引、撮合、定价、货值验证、谈判和非美元/低美元依赖结算，
推动真实贸易成交，并复用现有 1.0 贸易 SaaS 与供应链金融底座。

## 2. 现有基础
- 已有 IBC 1.0 贸易 SaaS
- 已有单证合一管理
- 已有供应链金融模块
- 已接入 Kun 坤支付聚合支付平台
- 现阶段不是推翻底座，而是在原有能力上构建 AI 商流平台层

## 3. 新系统目标
### 3.1 全球招引
吸引全球想与中国交易的合作伙伴、资源方、制造方、渠道商、采购商入驻。

### 3.2 AI 商流
通过 AI 完成企业画像、供需匹配、定价辅助、货值验证、谈判辅助和成交推进。

### 3.3 非美元结算
降低美元路径依赖，支持法币、稳定币、聚合支付、托管支付、组合结算等路径。

### 3.4 闭环履约
打通商机 → 报价 → 订单 → 合同 → 单证 → 履约 → 支付 → 融资闭环。

## 4. 系统分层
1. 全球招引前台
2. AI 商流中台
3. 交易履约中台（复用 1.0）
4. 支付金融底座
5. 风控与合规中台
6. 运营后台与 BI
