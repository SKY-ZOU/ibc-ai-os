## 升级清单

### 1. 视觉升级
- [ ] 首页全新 UI 设计（Hero 区域 + Features + 信任模块）
- [ ] 现代化配色：深海蓝 + 金色点缀 + 渐变
- [ ] 动效增强：入场动画、hover 交互、微动效
- [ ] 响应式优化：移动端体验提升

### 2. 内容升级
- [ ] 英文文案国际化打磨（landing page 英文更专业）
- [ ] 新增「合作伙伴/合作案例」展示模块
- [ ] 新增「信任资质」模块（资质证书、合规背书）
- [ ] 首页结构优化：Hero → 核心数据 → 功能 → 案例 → CTA → Footer

### 3. 功能扩展
- [ ] 新增「国别馆」入口卡片（点击跳转 /countries 或占位页）
- [ ] 新增「热门商机」轮播/列表
- [ ] 移动端底部导航栏

### 4. 技术优化（如有必要）

- [ ] 字体加载优化
- [ ] 性能检查
