# TASK-007 撮合工作台

## 目标
实现人工撮合界面 + 消息中心 + 转报价/转线索功能

## 依赖
- portal (前端)
- ai-trade-agent (后端 API)
- trade-saas-adapter (原有系统对接)

## 功能清单

### 1. 商机池管理
- 列表展示所有 Opportunity（供给/需求）
- 筛选：状态 / 国家 / 行业 / 类型
- 状态：待匹配 / 匹配中 / 已成交 / 已关闭

### 2. 撮合工作台
- 选择供需双方，生成匹配建议
- 展示匹配度评分
- 一键发起撮合

### 3. 消息中心
- 撮合过程中的沟通记录
- AI 辅助生成回复建议

### 4. 转报价 / 转线索
- 将商机转为正式报价单
- 将商机转给其他企业（线索分发）

## 页面设计 (Portal)
- `/opportunities` — 商机池列表
- `/opportunities/[id]` — 商机详情 + 撮合面板
- `/messages` — 消息中心

## 技术要求
- Portal 提供前端页面
- ai-trade-agent 提供匹配 API
- 调用 trade-saas-adapter 的报价/线索接口

## 验收标准
- [ ] 商机列表可展示
- [ ] 筛选功能可用
- [ ] 撮合面板可发起匹配
- [ ] 转报价功能可调用
