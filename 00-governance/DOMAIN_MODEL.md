# Domain Model

## 核心对象

### 1. 企业（Enterprise）
- 企业ID
- 名称
- 国家/地区
- 企业类型
- 资质文件
- 联系方式
- 支付偏好
- 风险等级
- 画像标签

### 2. 商品（Product）
- 商品ID
- 名称
- 品类
- 规格
- HS编码
- 原产地
- 质量标准
- 参考价格

### 3. 需求（Demand）
- 需求ID
- 采购方
- 品类
- 规格
- 数量
- 预算区间
- 交付地点
- 付款方式
- 交付时间
- 替代品接受度

### 4. 商机（Opportunity）
- 商机ID
- 来源
- 匹配双方
- 匹配分
- 当前阶段
- 跟进记录
- 意向等级
- 失败原因

### 5. 报价（Quotation）
- 报价ID
- 报价方
- 价格区间
- 价格依据
- 有效期
- 付款条件

### 6. 订单（Order）
- 订单ID
- 双方主体
- 商品明细
- 数量
- 金额
- 交付条件
- 支付路径

### 7. 合同（Contract）
- 合同ID
- 合同条款
- 签署状态
- 关联合同文件
- 风险提示

### 8. 单证（Document）
- 单证ID
- 单证类型
- 文件地址
- 关联合同/订单
- 验证状态

### 9. 资金（FundFlow）
- 支付单ID
- 支付路径
- 钱包/账户
- 金额
- 币种
- 结算状态

### 10. 资产（Asset）
- 资产ID
- 类型（仓单/提单/应收）
- 权属状态
- 抵押状态
- 融资状态
